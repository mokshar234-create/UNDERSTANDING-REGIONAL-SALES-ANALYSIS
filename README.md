📊 Understanding Regional Sales Analysis (Power BI Project)
📌 Overview

This project focuses on analyzing regional sales performance across the United States using Power BI.
The dashboard provides interactive visual insights to help understand sales trends, regional performance, and business growth opportunities.

🎯 Objectives
Analyze sales distribution across different regions
Identify top-performing and underperforming regions
Track sales trends over time
Understand product category performance
Support data-driven decision-making

🛠️ Tool Used
Power BI Desktop 📊
